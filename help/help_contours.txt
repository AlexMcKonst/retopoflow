Contours Help (press ? to hide help):

DRAWING:
- LMB: draw strip on surface
- SELECTMOUSE: select strip, junction, junction handle, or patch
- Shift + SELECTMOUSE: select multiple strips

- F: adjust brush size

TRANSFORM:
- G: grab
- R: rotate
- S: scale

MERGE / DELETE / OTHER:
- X: delete

- N: force new segment

FILL / SEGMENTS:
- SHIFT+F: fill selected strips w/ patch
- CTRL - +: adjust segments on strip

TIPS:
- Extend an existing geometry edgeloops by selecting it in Edit Mode before starting Contours.
