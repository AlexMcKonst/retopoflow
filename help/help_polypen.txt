Polypen Help (click to hide help):

DRAWING:
- LMB: insert vertex
- SELECTMOUSE: select vertex, edge, or face
- A: deselect all

MERGE / DELETE / DISSOLVE:
- D: dissolve selected edge
- X: delete selection

CONFIRM / CANCEL:
- ENTER / RETURN: confirm changes, return to Edit Mode
- ESC: cancel operation

TIPs:
- Inserting vertices is dependent on your selection:
-- select an edge then LMB to create a triangle from that edge, LMB again to turn it into a quad
